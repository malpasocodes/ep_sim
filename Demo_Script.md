# Demo Script – Earnings Premium Simulation

## Audience
Policymakers, journalists, or institutional leaders unfamiliar with the underlying methodology.

## Duration
~5 minutes live walkthrough.

## Outline
1. **Intro (30s)**
   - “This tool uses synthetic data to show how statewide vs. local earnings benchmarks change pass/fail outcomes.”
   - Mention it’s a public demo (MIT-licensed code on GitHub, deployed on Render).

2. **State & Simulation Overview (1 min)**
   - Select a large state (e.g., California) and call out the summary card metrics.
   - Explain how CZ counts scale with state size using the new mapping.

3. **Local Earnings Histogram (1 min)**
   - Point out bars vs. the statewide dashed line.
   - Adjust the inequality slider to show the spread widening/narrowing.

4. **Main Charts (1.5 min)**
   - Scatterplot: walk through quadrants and what “Pass Local Only” means.
   - Bar chart: highlight how many programs only pass locally and why statewide metrics misclassify them.

5. **Narrative & Sample Programs (1 min)**
   - Read the narrative snippet emphasizing ranges and classification share.
   - Scroll to “Example Program Trajectories” and compare early vs. later bars.
   - Note that several programs just above the benchmark later were marginal earlier.

6. **Call to Action (30s)**
   - Encourage the audience to explore other states or download the code (MIT license) to extend the prototype.
   - Mention planned next steps (real data integration, additional filters).
